    DO ONE THING AND DO IT WELL

    We LOVE D.R.Y.

    We have Apache Commons in Java, so JavaScript?

    When you have a common problem, parse date string for example, what you will do?

    Write a snippet util code? "Call" Google? Or you only need is a stable utility lib, then happy life together forever...

    So where to find those kind of util libs? You might know some of those, but why not we collect them all and let everyone know about them here?

    The Show Time

    • string.js
    • Numeral.js
    • Moment.js - Parse, validate, manipulate, and format dates.
    • Chance.js - Generator of random strings, numbers.
    • jsUri

    • CryptoJS - MD5, AES...

    • Underscore.js - map, each, bind, pick, mixin.

    • Backbone.js - Gives structure to web applications by providing MV*.

    • RequireJS - A JavaScript file and (AMD) module loader.

    • jQueryUI Widget Factory - Stateful UI components base structure

    • Bootstrap - A powerful front-end framework for faster and easier web development.
